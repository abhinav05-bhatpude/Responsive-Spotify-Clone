# Responsive Spotify Clone

A responsive Spotify-inspired music player UI built using HTML and CSS.

## About

This project recreates Spotify's user interface using only HTML and CSS. It focuses on responsive design and modern UI styling.

## Features

- Spotify-inspired design
- Responsive layout
- Sidebar navigation
- Music cards section
- Pure HTML and CSS

- ## Tech Stack

- HTML5
- CSS3
